## What Are CSS Gradients?

CSS gradients let you display smooth transitions between colors without using image files. They are generated by the browser at render time, which means they scale perfectly to any size and add zero HTTP requests to your page.

Modern CSS supports three types of gradients: linear, radial, and conic.

## Linear Gradients

A linear gradient transitions colors along a straight line. The simplest form specifies two colors:

```css
background: linear-gradient(to right, #f97316, #6366f1);
```

The first argument sets the direction. You can use keywords (`to right`, `to bottom left`) or angles (`45deg`, `135deg`). The remaining arguments are color stops.

You can add as many color stops as you want:

```css
background: linear-gradient(135deg, #f97316 0%, #ec4899 50%, #6366f1 100%);
```

Each color stop can include a position (percentage or pixel value) that controls where that color is centered in the gradient.

## Radial Gradients

A radial gradient radiates outward from a center point in an ellipse or circle shape.

```css
background: radial-gradient(circle at center, #f97316, #1a1a2e);
```

You can control the shape (`circle` or `ellipse`), the size (`closest-side`, `farthest-corner`), and the position of the center point.

## Conic Gradients

A conic gradient rotates color stops around a center point, like the segments of a pie chart.

```css
background: conic-gradient(from 0deg, #f97316, #6366f1, #f97316);
```

Conic gradients are useful for pie charts, color wheels, and angular loading indicators.

## Layering Multiple Gradients

CSS backgrounds accept multiple layers separated by commas. You can combine gradients with images and other gradients:

```css
background:
  linear-gradient(rgba(0,0,0,0.5), rgba(0,0,0,0.5)),
  linear-gradient(45deg, #f97316, #6366f1);
```
